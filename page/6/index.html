elf file OS ABI invalid

发表于 2013-08-28
昨天调用第三方库的JNI时发现如下问题： elf file OS ABI invalid 问题出自ldd，先看一下ldd的版本号。使用： ldd –version 这个问题是因为在ldd 为2.12的机器上编译。拿到ldd 为2.5的机器上去运行加载，然后就挂掉了。解决方法就是使用ldd 2.5的机器编译。 JNI的问题一般出自OS不兼容，gcc版本，glibc版本（libc.so.6）,ldd版本( ld-x.xx.so )不一致。
