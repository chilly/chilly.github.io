根基类的static先被初始化。然后到子类的构造函数，在到基类的构造函数。

每一个类都有自己的方法。超类的和子类的方法名必须相同。但是超类可以不实现那个方法。
最后在某一个类中用一个函数来用超类的引用当作参数。调用和子类同名的函数叫做多态。
这样做的好处是如果超类和子类都存放在同一个数组里，这样就可以节省代码。

抽象类中不需要每一个方法都是抽象的。但至少有一个。
