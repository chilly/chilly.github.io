KMeans 算法

发表于 2009-04-23 | 更新于 2017-06-29 | |

KMeans最主要是思想是

  1. 播撒K个标识点，K是你要分成几类。
  2. 网络中所有点对这K个点计算距离，并归入最近的一个标识点中。这样就得到K个类
  3. 重新计算这K个类的中心距离。
  4. 是否要更新K个类的中心距离，是否迭代次数为0，如果要更新标识点的位置，并且迭代次数不为0，更新标识点，并跳转回2.
